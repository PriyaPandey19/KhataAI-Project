# 📒 KhataAI - Smart Digital Ledger with AI Insights

> An AI-powered digital ledger application that helps small businesses and shopkeepers manage customer transactions, track payments, analyze financial data, and gain intelligent business insights.


---

## 🚀 Overview

KhataAI is a modern digital bookkeeping platform built for small businesses. It replaces traditional paper-based account books with a secure web application that not only records customer transactions but also provides AI-powered financial insights and analytics to help business owners make smarter decisions.

---

## ✨ Features

### 👥 Customer Management
- Add, edit and delete customers
- View complete customer profiles
- Search customers instantly

### 💰 Transaction Management
- Record credit and debit transactions
- View transaction history
- Automatic balance calculation
- Payment tracking

### ⏰ Smart Reminders
- Create payment reminders
- Track pending dues
- Reminder status management

### 📊 Analytics Dashboard
- Income vs Expense analysis
- Revenue visualization
- Financial summary
- Customer analytics
- Interactive charts

### 🤖 AI Features
- Smart financial insights
- Customer risk analysis
- Spending pattern analysis
- Business recommendations
- Predictive analytics

### 🔐 Authentication
- Secure JWT Authentication
- User Registration
- Login & Logout
- Protected Routes

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/PriyaPandey19/KhataAI-Project.git
```

```bash
cd KhataAI-Project
```

## Install Dependencies

### Backend

```bash
cd server
npm install
```

### Frontend

```bash
cd client
npm install
```

### ML Server

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Project

### Backend

```bash
npm start
```

### Frontend

```bash
npm run dev
```

### ML Server

```bash
python main.py
```
